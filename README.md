# 📝 Angular 19 Todo App

A fully-featured todo application built with **Angular 19**, featuring signals, local storage persistence, and a modern responsive UI.

![Angular](https://img.shields.io/badge/Angular-19-DD0031?logo=angular)
![TypeScript](https://img.shields.io/badge/TypeScript-5.x-3178C6?logo=typescript)
![License](https://img.shields.io/badge/license-MIT-green)

## ✨ Features

- ✅ Add, edit, delete tasks
- ✅ Mark tasks as complete/incomplete
- ✅ **Priority levels** (Low, Medium, High) with color coding
- ✅ **Due dates** with overdue highlighting
- ✅ **Search tasks** by title
- ✅ **Sort options** (by newest, priority, due date)
- ✅ **Filter tasks** (All, Active, Completed)
- ✅ **Undo delete** (5-second window to restore)
- ✅ **Local storage persistence** (tasks survive page refresh)
- ✅ Real-time task counter & statistics
- ✅ Responsive design (mobile-friendly)
- ✅ Smooth animations

## 🚀 Quick Start

### Prerequisites
- Node.js (v18 or later)
- Angular CLI (`npm install -g @angular/cli`)

### Installation

```bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/todo-app-angular.git
cd todo-app-angular

# Install dependencies
npm install

# Run the development server
ng serve
